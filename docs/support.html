Javasta サポート

Java SE 資格試験対策アプリ「Javasta」のサポートページです。

よくある質問

🔒 Gold レベルや模擬試験が利用できません

Gold レベルと Silver 模擬試験はプレミアムプラン（¥980 買い切り）でご利用いただけます。 設定画面の「プレミアム」セクションから購入できます。

📱 機種変更したらデータが消えました

iCloud 同期が有効な場合、iCloud にバックアップされたデータが自動で復元されます。 設定 → Apple アカウント → iCloud → Javasta がオンになっているかご確認ください。

💳 購入を復元したい

アプリ内の「設定 → プレミアム → 購入を復元する」をタップしてください。 同じ Apple ID でサインインしていれば無料で復元できます。

🔔 通知が届きません

iPhone の「設定 → 通知 → Javasta」で通知が許可されているかご確認ください。 アプリ内「設定 → 通知」でリマインダー時刻も変更できます。

📝 問題の誤りを見つけました

問題画面の「問題の不備を報告」ボタン、または下記メールにてご連絡ください。 確認のうえ修正いたします。
