プライバシーポリシー

最終更新日: 2026年5月27日

Javasta（以下「本アプリ」）は、茂木史明（以下「開発者」）が提供する Java SE 資格試験対策アプリです。 本ポリシーは、本アプリがどのような情報を扱うかを説明します。

1. 収集する情報

本アプリは、個人を特定できる情報を収集しません。
アプリの動作に必要なデータ（学習進捗・回答履歴・設定など）は、 お客様のデバイス内または iCloud（Key-Value Storage）にのみ保存されます。 これらのデータが開発者のサーバーに送信されることはありません。

2. 利用目的

  • 学習進捗・正答率・連続学習日数などの表示
  • 復習キュー・ブックマークの管理
  • iCloud を通じた複数デバイス間のデータ同期
  • ホーム画面ウィジェットへのデータ表示

3. 第三者への情報提供

開発者は、お客様のデータを第三者に販売・提供・開示しません。

4. アナリティクス・トラッキング

本アプリは広告ネットワークやサードパーティのアナリティクス SDK を使用していません。 App Store のみを通じて配信しており、Apple の標準的な購入・クラッシュレポートの仕組みのみを利用します。

5. アプリ内課金

プレミアムプランの購入は Apple の App Store を通じて処理されます。 支払い情報は Apple が管理し、開発者はカード情報等にアクセスできません。 Apple のプライバシーポリシーは こちら をご参照ください。

6. 子どものプライバシー

本アプリは 13 歳未満を対象としておらず、そのような利用者から意図的に情報を収集しません。

7. ポリシーの変更

本ポリシーを変更する場合は、このページに掲載します。 重要な変更がある場合はアプリのアップデートでお知らせします。
