# Javasta Launch Marketing Plan

## Positioning

Javasta should be sold as a Java Silver / Gold exam app for learners who want to understand code execution, not only memorize answers.

Primary message:

> Java試験の出力問題を、コードの流れで理解する。

Secondary messages:

- Silver / Gold対応
- 通常練習、復習、20分模試、本番模試
- 各問題にコード追跡型の解説
- 問題報告とフィードバック導線で継続改善

## Implemented Launch Hooks

- Problem issue reports from quiz result screens.
- General feedback mail from Settings.
- Shareable practice and mock exam result text.
- Manual review request from Settings.
- Debug-only release readiness dashboard.
- Content quality audits for generic explanations, repeated wording, duplicate code, duplicate choice sets, and validation issues.

## App Store Page

Suggested screenshot order:

1. Java Silver / Gold 対策
2. コード問題
3. コード追跡解説
4. 20分模試・本番模試
5. 復習と弱点管理
6. 用語集・学習マップ
7. 問題報告・改善導線

## Ads

Start small with App Store search intent keywords:

- Java Silver
- Java Gold
- Java SE 17
- Java 資格
- Java 問題集
- Java 模擬試験
- Java Stream API
- Java ラムダ式
- Java 出力問題

Best early campaign:

1. Exact-match Apple Search Ads for `Java Silver` and `Java Gold`.
2. Small daily budget.
3. Custom product pages later for Silver, Gold, and mock-exam intent.
4. Compare conversion by App Store Connect acquisition source.

## Pre-Sale Checklist

- Set final App Store URL in share text after release.
- Decide paid model: Silver, Gold, and bundle, or single paid unlock.
- Add StoreKit product IDs when pricing is finalized.
- Prepare privacy policy and support URL.
- Capture localized App Store screenshots.
- Run `QuestionBank.contentQualityIssues()` and `QuestionBank.validationIssues()` before every release.
